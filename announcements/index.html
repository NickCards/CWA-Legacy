CWA Legacy Alpha Testing

CWA Legacy is now open for Alpha Testing!

What This Phase Is About

The primary goal of this testing phase is to evaluate server stability, performance, and database reliability under real-world conditions with multiple players online at the same time.

While friends have been helping test CWA Legacy for several months, opening the server to the public will allow us to better understand how the Login Server, Game Server, and Database perform under increased load and concurrent player activity.

During Alpha Testing, players may encounter:

  • Server crashes
  • Client crashes
  • Unexpected bugs
  • Missing or incomplete features
  • Balance issues
  • Data inconsistencies
  • Temporary downtime for maintenance or updates

Although player progression is currently saved to the database and preserving player data is a major goal of the project, there may be situations during Alpha Testing where player data must be adjusted, repaired, or reset while major systems are being developed.

We appreciate your patience and feedback as we continue improving the experience.

Launcher Security Notice

Unsigned executable & SmartScreen

The CWA Legacy Launcher is currently distributed as an unsigned executable. Because it is not code-signed with a trusted certificate, Windows may display a Microsoft Defender SmartScreen warning when downloading or launching the application.

This warning is expected and does not indicate that the launcher contains malware. As the project grows, we may explore obtaining a code-signing certificate to improve the installation experience and reduce security warnings.

Always download the launcher directly from the official CWA Legacy website.

How You Can Help

Players can assist by:

  • Reporting bugs and crashes
  • Testing new features
  • Stress-testing server systems
  • Providing feedback on progression and balance
  • Suggesting improvements and future features

Every login, minigame, purchase, profile update, and social interaction helps us better understand how the server performs and what needs improvement. All of this can be sent directly to me on Discord @nickcards.

Thank you for helping shape the future of CWA Legacy.
