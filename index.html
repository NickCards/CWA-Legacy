What is CWA Legacy?

A public Oxide-based server with a dedicated database and login server that saves your progression, purchases, and characters.

Download Launcher View Roadmap

A Server Built to Last

CWA Legacy is a fully public Oxide server running alongside a dedicated Database and Login Server. This setup allows us to save player progression, purchases, character customizations, friends list, profiles, and soon Houses & Lots!

It also lets us assign a player to an account — so when you create a character, it's yours.

Another major benefit of this architecture is that it allows us to heavily modify or update the Oxide server without losing player progression or account data. As CWA Legacy continues to evolve, we will do our best to preserve existing accounts, characters, and player progress whenever new features, content, or server updates are introduced.

Thank You, Oxide

None of this would be possible without the work of the Oxide team, and we are incredibly grateful for everything they have accomplished. Their efforts have provided the foundation that makes projects like CWA Legacy possible.

As new additions, improvements, and features are developed by the Oxide team and the wider community, we will continue evaluating and integrating them into CWA Legacy whenever possible.
