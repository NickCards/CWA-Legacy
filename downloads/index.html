Getting Started Guide

Installation & Setup

  • After installing the CWA Legacy Launcher, you'll be prompted to select your Clone Wars Adventures client folder.
  • The selected folder must contain the original CloneWars.exe file. The launcher automatically verifies the file path and notifies you if the executable can't be found.
  • If you use another Oxide-based server, we recommend creating a separate copy of your client files for CWA Legacy.
  • Once the launcher points to a valid client folder, it will guide you through account creation and login.

First Login & Character Creation

Your account username is not your in-game display name — similar to how the original Clone Wars Adventures account system worked.

After creating your account, setting your password, and configuring your security questions, you'll be brought to the character creation screen.

Character Creation Options

  • First Name (Required)
  • Last Name (Optional)
  • Gender
  • Species

Important Notes

  • Species can be changed later in-game.
  • Gender cannot currently be changed after character creation.
  • The Clone species is only available with the Male gender option. (CWA predates The Bad Batch.)
  • After entering the game, you can further customize your character through the in-game customization menu.
  • Character customization costs credits; however, all new players start with enough credits to fully customize their character.

Purchases

Known Issues

Store purchases are functioning and items are successfully added to your inventory. After completing a purchase, you'll see a "Processing" screen that may remain for 1–2 minutes before showing an error message. Although the error appears, the purchased item is still successfully delivered to your inventory. This issue is being investigated.
